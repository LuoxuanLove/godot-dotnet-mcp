# 目录树与文件职责

本文档从目录树视角说明 `addons/godot_dotnet_mcp` 的主要文件结构和职责分布，帮助维护者快速定位实现入口。

---

## 顶层目录

```text
addons/godot_dotnet_mcp
├─ custom_tools/
├─ docs/
├─ localization/
├─ plugin/
├─ release/
├─ tools/
├─ ui/
├─ CHANGELOG.md
├─ CHANGELOG.zh-CN.md
├─ plugin.cfg
├─ plugin.gd
├─ README.md
└─ README.zh-CN.md
```

---

## 顶层文件

| 路径 | 作用 |
|---|---|
| `plugin.gd` | 插件入口、生命周期装配、Dock 与服务桥接 |
| `plugin.cfg` | Godot 插件声明与版本信息 |
| `README.md` / `README.zh-CN.md` | 对外使用与接入说明 |
| `CHANGELOG.md` / `CHANGELOG.zh-CN.md` | 版本增量记录（中英双份） |
| `.gitignore` | 忽略规则，包括 `.backup/` 等运行期产物 |

## `docs/`

当前文档按四层组织：

- `架构/`
- `界面/`
- `模块/`
- `附录/`

文档本身也是项目的一部分，需要和代码同步维护。

## `localization/`

### 关键文件

| 路径 | 作用 |
|---|---|
| `localization_service.gd` | 语言字典装载、当前语言切换、文本读取 |
| `locale_en.gd` | 英文基线字典 |
| `locale_zh_cn.gd` | 简体中文 |
| `locale_zh_tw.gd` | 繁体中文 |
| `locale_ja.gd` | 日文 |
| 其它 `locale_*.gd` | 德语、西语、法语、葡语、俄语等语言资源 |

### `tools/intelligence/`

Intelligence 域目录结构比其他域更复杂，包含 6 个文件：

| 文件 | 职责 |
|---|---|
| `executor.gd` | 调度器：初始化 4 个 impl，扫描加载 `custom_tools/`，路由 `execute` 调用 |
| `atomic_bridge.gd` | 原子桥：`call_atomic(executor_name, tool_name, args)` 动态加载下层 executor；包含写保护逻辑（阻止修改 `res://addons/godot_dotnet_mcp/` 内置目录，`custom_tools/` 除外） |
| `impl_project.gd` | 实现 `project_state` / `project_advise` / `project_configure` / `project_run` / `project_stop` / `runtime_diagnose` |
| `impl_scene.gd` | 实现 `scene_validate` / `scene_analyze` / `scene_patch` |
| `impl_script.gd` | 实现 `bindings_audit` / `script_analyze` / `script_patch` |
| `impl_index.gd` | 实现 `project_index_build` / `project_symbol_search` / `scene_dependency_graph` |

## `ui/`

| 路径 | 作用 |
|---|---|
| `mcp_dock.tscn` / `mcp_dock.gd` | Dock 根场景与控制器 |
| `server_panel.tscn` / `server_tab.gd` | Server 页签 |
| `tools_tab.tscn` / `tools_tab.gd` | Tools 页签 |
| `config_panel.tscn` / `config_tab.gd` | Config 页签 |

当前 UI 已场景化，`plugin.gd` 不再动态拼整个 Dock。

---

## `custom_tools/`

这是 User 工具目录。典型内容包括：

- 用户脚本
- 自进化脚手架生成结果
- 删除前备份目录 `.backup/`

这部分内容属于运行中可变区域，不应与内置工具域混写。

---

## `release/`

用于承载对外发布相关内容。它不参与插件运行时装配，但影响打包和发布工作流。

---

## 不需要重点关注的文件

以下文件通常不需要在功能维护时单独阅读：

- `*.uid`
- 纯资源 ID 映射文件
- 未被引用的旧测试夹具

但若出现资源失联或 Godot 资源重定向问题，仍需要连同 `.tscn/.gd/.uid` 一起检查。
